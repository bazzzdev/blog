# Медленный веб, который приятно читать

Есть сайты, которые пытаются доказать свою современность количеством движущихся частей. А есть сайты, которые работают тише и поэтому кажутся взрослее.

## Что делает чтение спокойным

- хороший ритм заголовков;
- предсказуемая навигация;
- достаточная ширина колонок;
- контраст без агрессии.

> Хороший интерфейс не отвлекает от мысли, а поддерживает ее.

### Минимальный стек, максимальное внимание к деталям

Когда каркас прост, начинает играть роль уже не количество блоков, а качество решений вокруг них.

| Прием | Зачем он нужен |
| --- | --- |
| Воздух вокруг текста | Глазу проще держать строку |
| Сдержанная анимация | Интерфейс ощущается живым, но не шумным |
| Сильная типографика | Даже список постов выглядит законченным |

#### Код тоже должен читаться спокойно

```ts
const readingMode = {
  motion: "restrained",
  palette: "warm-paper",
  typography: "editorial",
};
```

И если хочется оставить себе маленькую заметку[^1], footnotes тоже должны выглядеть естественно.

[^1]: В блоге про текст мелкие детали имеют непропорционально большое значение.
